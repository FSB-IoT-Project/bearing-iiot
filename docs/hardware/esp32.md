# Thiết bị ESP32 S3 N16R8

ESP32 S3 N16R8 đóng vai trò là thiết bị đầu cuối thu thập dữ liệu từ các cảm biến.

## Thông số kỹ thuật
- **MCU**: Xtensa® Dual-Core 32-bit LX6
- **Connectivity**: Wi-Fi 802.11 b/g/n, Bluetooth v4.2 BR/EDR & BLE
- **Sensor support**: I2C, SPI, ADC, UART

## Vai trò trong hệ thống
- Đọc dữ liệu từ cảm biến gia tốc (vibration) và nhiệt độ.
- Tiền xử lý dữ liệu thô (filtering).
- Gửi dữ liệu về Gateway thông qua giao thức MQTT/LoRa.
